# jnsilva.com

Personal portfolio site for **JN Silva** — photographer, filmmaker, and creative director.

## Stack

- **Next.js 16** (App Router)
- **React 19**
- **Tailwind CSS v4**
- **Three.js + React Three Fiber** (sacred geometry visualizations)
- **GSAP** (scroll-driven animations, parallax)
- **TypeScript**

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Deployed on [Vercel](https://vercel.com). Push to `main` triggers automatic deployment.
